# Stock Up Dinners — Pinterest Pin Strategy

CEO Agent, 2026-04-23.

Pinterest is the highest-ROI evergreen channel for food/meal prep content. Pins drive traffic for months after posting. This doc covers the full launch pin set (12 pins), posting cadence, board structure, and SEO keyword strategy.

All pins link to the Beehiiv landing page (stockupdinners.com). Set up this content AFTER Task 8 (Beehiiv) is live.

---

## Board Structure (Create These First)

Create a Pinterest Business account and set up these boards before pinning:

| Board Name | Description | Primary Audience |
|---|---|---|
| **Costco Meal Planning** | Weekly dinner plans built from Costco staples | Costco shoppers, budget-conscious families |
| **Family Dinner Ideas** | Easy weeknight dinners for busy families | Moms, family meal planners |
| **Meal Prep Made Simple** | Batch cooking, prep-day workflows, storage tips | Meal preppers, time-savers |
| **Budget Grocery Hacks** | Save money at the warehouse store | Budget shoppers |

## SEO Keywords for All Pins

Include these in pin titles and descriptions. Pinterest search works like Google — keywords matter.

**Primary:** "Costco meal plan," "Costco dinner ideas," "warehouse club meal prep"
**Secondary:** "family dinner plan," "weekly meal prep Costco," "Costco staples recipes," "cheap family dinners"
**Long-tail:** "14 dinners from one Costco trip," "Costco meal prep beginners," "how to meal plan with Costco"
